# 🧱 市集擺攤陳列模擬器

輸入桌子尺寸、建立自己的陳列道具，在 2D 俯視圖上拖曳擺位，右側即時同步 3D 預覽——出攤前先在電腦上把桌面配置演練一遍。

## 功能特色

- **自訂桌子尺寸**：輸入寬／深／高（cm），畫面依實際比例呈現
- **物件庫**：建立與編輯自己的商品與道具（名稱、寬高深、顏色），支援三種形狀：
  - 方塊：一般商品、盒子、箱子
  - 階梯展示架：可自訂 2–8 層，3D 會呈現實際階梯造型
  - 洞洞板：直立板身，厚度固定 2cm
- **任意角度旋轉**：拖曳物件上方的旋轉把手可轉到任何角度（自動吸附 15°，按住 `Shift` 可 1° 微調），也可以直接在工具列輸入度數
- **自動堆疊**：把物件拖到另一個物件上方就會自動疊上去，拖開就落回桌面；放在階梯展示架上會停在對應層的高度，2D 圖會顯示「↑15cm」之類的高度標籤
- **防呆警示**：物件超出桌面時，2D 與 3D 同步變紅提醒
- **3D 預覽**：拖曳旋轉視角、滾輪縮放，模擬客人站在攤位前看到的樣子
- **場景模擬**：可調整背景與地板顏色、桌布開關／顏色／下擺長度，並可放一個可調身高的木人偶站在攤位前，感受實際比例
- **自訂版面**：拖曳分隔條調整側欄與 2D／3D 區的寬度（設定會記住）
- **手機也能用**：小螢幕自動切換為單一視圖（2D／3D 切換鈕）＋抽屜式選單，支援觸控拖曳與雙指縮放
- **自動存檔**：配置自動存在瀏覽器中，關掉重開不會消失

## 開始使用

### 方法一：直接在線上使用
前往 https://pdatone.github.io/market-product-display-helper/

### 方法二：下載到自己電腦執行

因為頁面使用了 ES Modules，不能直接雙擊 HTML 檔開啟，需要一個本機靜態伺服器：

```bash
git clone https://github.com/pdatone/market-product-display-helper.git
cd market-product-display-helper
python3 -m http.server 8123
```

然後用瀏覽器打開 `http://localhost:8123`。

- macOS 內建 Python 3，直接可用
- Windows 使用者可安裝 [Python](https://www.python.org/downloads/)，或改用 VS Code 的 Live Server 擴充功能
- 若出現 `Address already in use`，把 `8123` 換成其他數字（例如 `8124`）再試

## 操作說明

| 操作 | 方式 |
|------|------|
| 設定桌子 | 左側輸入寬／深／高，按「套用桌子尺寸」 |
| 建立道具 | 左側表單填名稱、寬高深、形狀、顏色，按「＋ 新增到物件庫」 |
| 編輯道具 | 點物件庫項目旁的 ✎，修改後按「儲存修改」；桌上已擺放的同款物件會立即套用 |
| 擺上桌面 | 點物件旁的「加入」，會自動找空位放置 |
| 移動 | 在 2D 俯視圖直接拖曳（自動吸附 1cm 格點） |
| 旋轉 | 拖曳選取物件上方的圓形把手（吸附 15°，`Shift` 微調 1°），或在工具列輸入角度；`R` 鍵一次轉 90° |
| 堆疊 | 直接把物件拖到另一個物件（或階梯展示架）上方 |
| 複製／刪除 | 先點選物件，再按工具列按鈕或快捷鍵 `D`／`Del` |
| 3D 視角 | 在右側預覽區拖曳旋轉、滾輪縮放 |
| 手機版 | 左下 ☰ 開啟選單，底部按鈕切換 2D／3D，雙指縮放 3D 視角 |

## 小技巧

- 疊放順序是「最後移動的在最上面」：如果疊的結果不如預期，把想放上面的那個再拖一下就好
- 移動下層物件時，疊在上面的東西不會跟著走（會落回桌面），搬動整疊時記得由下往上重新擺

## 常見問題

**我的配置存在哪裡？**
存在瀏覽器的 localStorage，只在同一台裝置、同一個瀏覽器有效。清除瀏覽器網站資料會一併清掉。

**可以把配置傳給朋友嗎？**
目前還不行，匯出／匯入功能在規劃中。
